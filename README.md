# أذكار المسلم اليومية

تطبيق فلاتر بسيط باللغة العربية يجمع أبرز الأذكار اليومية للمسلم في أقسام مرتبة مثل أذكار الصباح والمساء، أذكار النوم، وأذكار الطعام. يوفر التطبيق واجهة عربية بالكامل مع إمكانية البحث داخل جميع الأذكار للوصول السريع إلى المطلوب.

## المتطلبات
- Flutter 3.13 أو أحدث
- Dart SDK 3.2 أو أحدث

## التشغيل
بعد استنساخ المشروع لأول مرة، قم بتوليد ملفات ومنصات فلاتر الافتراضية:

```bash
flutter create .
```

ثم ثبّت الاعتمادات وشغّل التطبيق:

```bash
flutter pub get
flutter run
```

## إنشاء ملف APK للتنزيل
للحصول على ملف APK جاهز للتثبيت دون الحاجة إلى بيئة تطوير محلية، يمكنك تشغيل سير العمل المدمج في GitHub Actions:

1. افتح صفحة المستودع على GitHub.
2. انتقل إلى تبويب **Actions** واختر سير العمل **Build Android APK**.
3. اضغط على زر **Run workflow**، ويمكنك (اختياريًا) إدخال اسم وسم مثل `v1.0.0` إذا رغبت في التحكم باسم الإصدار.
4. انتظر حتى يكتمل التشغيل بنجاح.
5. ستجد ملف `app-release.apk` ضمن قسم **Artifacts** في نفس صفحة التشغيل لتنزيله مباشرةً.
6. بعد كل تشغيل ناجح، سيُنشئ GitHub إصدارًا جديدًا تلقائيًا ويحمل الملف نفسه، سواءً أدخلت وسمًا أم لا (عند عدم تحديد وسم سيُستخدَم اسم تلقائي مثل `auto-build-123456789`). يمكنك الوصول إلى أحدث ملف APK من تبويب **Releases** أو مباشرة عبر الرابط التالي:

   - [آخر إصدار متاح للتنزيل](https://github.com/karwansawer-art/DDD/releases/latest/download/app-release.apk)

> ملاحظة: يعتمد سير العمل على أمر `flutter create . --platforms=android` لإنشاء مجلدات منصة أندرويد قبل البناء، لذا لا حاجة لرفع ملفات المنصة إلى المستودع.

## المزايا
- واجهة عربية مع خط Cairo من Google Fonts.
- تبويبات تغطي الأذكار الأساسية اليومية (الصباح، المساء، بعد الصلاة، الاستيقاظ، النوم، الطعام وغيرها).
- إمكانية البحث في جميع الأذكار وعرض النتائج مباشرة.
- بطاقات مرتبة تعرض نص الذكر وعدد التكرار والمصدر عند توفره.
